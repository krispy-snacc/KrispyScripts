# Google Classroom - Show Inline Attachments

Displays post attachments inline in the Google Classroom stream feed without having to open each post.

## Features

-   **Inline Display**: Shows all attachments directly in the stream feed
-   **Native Styling**: Matches Google Classroom's native attachment UI
-   **Auto-Detection**: Automatically processes new posts as you scroll
-   **Persistent**: Works across page navigation and re-renders

## Installation

1. Install a userscript manager:

    - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Edge, Safari)
    - [Violentmonkey](https://violentmonkey.github.io/) (Chrome, Firefox, Edge)

2. Install the script:

    - Click [here](classroom_inline_attachments.js) to view the script
    - Click "Raw" and your userscript manager will prompt installation

3. Navigate to Google Classroom - attachments will appear automatically

## Usage

Once installed, visit any Google Classroom stream page. Attachments will appear inline below each post with file icons, names, and clickable links.

## Notes

-   Works with all file types (PDFs, docs, images, videos, etc.)
-   Handles Google Drive files and external links
-   No configuration required
